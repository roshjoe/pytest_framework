Building a pytest framework to test the following site https://practicetestautomation.com/logged-in-successfully/.
A positive login test case and a negative login test case has been created.
Created markers : login test , negative, positive 
The requirements.txt file lists the different versions of pytest , selenium , webdriver manager and HTML used.
created a .ini file to log the different markers used in the test case
The HTML report file is created to access the test reports on the browser.
The reports folder , pytest.ini & requirement.txt files are created one level outside the tests folder but within the python projects folder.
created fixtures file (conftest) to add the choice of browsers for test. Here the default browser chosen is Chrome.
